What is The Differnce b/w Programming language and Script?

  • Programming Language:

  • Definition: A formal system designed to communicate instructions to a computer. It provides a framework for writing all types of programs, from small scripts to large-scale applications.
    Examples: C, C++, Java, Python, Rust.


  • Programming Script:

  • Definition: A subset of programming languages designed for automating tasks, controlling systems, or adding functionality to existing software.
    Examples: Python, JavaScript, Bash, Perl, Ruby.


Some Points on P.L.(Programming Language) and P.S.(Programming Script)

Aspect Programming Language Programming Script
Purpose Build complete applications/systems. Automate tasks or control existing systems.
Execution   Can be compiled or interpreted. Mostly interpreted (run line-by-line).
Complexity    More complex witd advanced features.   Simpler and task-specific.
Performance    High-performance, optimized for speed. Typically slower, focusing on ease of use.
